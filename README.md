## What is Python Game Box?

Python Game Box (PGB) is a nostalgic tribute to the golden age of Flash games. It features a curated collection of classic mini-games, all written in **100% pure Python code**.

### Key Features

- **Pure Python**: All games are built from scratch using only Python and standard libraries
- **No Dependencies**: Runs without complex setup or external game engines
- **Lightweight**: Simple, fast, and easy to run
- **Nostalgic**: Brings back the fun of classic Flash-era mini-games

## Why Pure Python?

Every game in this collection is written entirely in Python without using game engines like Pygame, Unity, or Godot. This demonstrates what's possible with standard Python libraries and serves as a testament to the language's versatility.
